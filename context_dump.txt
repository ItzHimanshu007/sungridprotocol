# â˜€ï¸ SunGrid Protocol

**DeCentralized Peer-to-Peer Energy Trading Platform**

SunGrid Protocol enables direct energy trading between producers (solar owners) and consumers using blockchain technology. By removing intermediaries, we empower communities to trade renewable energy efficiently, transparently, and securely.

## ğŸš€ Quick Start

### Prerequisites
- Node.js & npm
- [Foundry](https://getfoundry.sh/) (for blockchain)
- [PostgreSQL](https://www.postgresql.org/) (optional, for API cache)

### 1. Start Local Blockchain
```bash
# Start Anvil
anvil --block-time 1
```

### 2. Deploy Contracts
```bash
cd packages/contracts
forge build
forge script script/Deploy.s.sol:DeployScript --rpc-url http://127.0.0.1:8545 --broadcast --skip-simulation
```

### 3. Start Frontend
```bash
cd apps/web
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to access the application.
